About

Hello! Thank you for checking out this website! This project was created by Johnny Po and Hanadi Alqatani for the class CS 458 Information Visualization at Oregon State University. This project aims to make the graduate statistics and data of OSU more accessible, as well as presenting it in a more visually appealing method. Through the use of color, gradients, bold fonts, charts, graphs and a data organization that makes more sense, users will be able to easily see and comprehend OSU's graduate statistics. The purpose of this project is to summarize and accompany the main OSU graduate statistics page.

All of the information that you find on this site originates from Oregon State University's own website on Graduate Summaries at institutionalresearch.oregonstate.edu

The full source code for this project can be found at the project's Github repository
